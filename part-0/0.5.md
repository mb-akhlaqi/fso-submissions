## Exercise 0.5: Single page app diagram
### Notable events that transpire between browser and server:

1. User opens SPA version of notes web page.
2. Browser requests `/spa` (GET).
3. Server returns the HTML document.
4. Browser requests `main.css`.
5. Broswer requests `spa.js`.
6. Browser executes the JS code.
7. Browser requests `data.json`.
8. Server returns list of notes.
9. Browser renders the notes.
